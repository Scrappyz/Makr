# Makr

A boilerplate code generator and manager.

## Development

### Requirements
- [Visual Studio](https://visualstudio.microsoft.com/)
- [.NET](https://dotnet.microsoft.com/en-us/download/dotnet/10.0)
    - Or install during Visual Studio installation
- [pnpm](https://pnpm.io/installation)
- Angular
    - Install with `pnpm`

### Setup

#### Running the Frontend
- Go to `frontend` directory.
- Run `pnpm install` (if you haven't before).
- Run `pnpm start`.

#### Running the Backend

##### Visual Studio
- Run application in `http`.
